Unsere Webseite über Strings und deren Methoden

String ist ein Datentyp zur Darstellung von Zeichenketten. String ist kein primitiver Datentyp da es von Objekt erbt. Die Stringklasse hat viele statische als auch objekt Methoden, welche das bearbeiten von Strings ermöglichen. Strings sind immutable, dass heißt die meisten Funktionen welche Veränderungen durchführen geben tatsächlich nur eine veränderte Kopie des Ausgangstrings wieder

length()

Diese Funktion gibt die länge eines Strings als int zurück, d.h. die Anzahl der Zeichen in dem String. Sei String s1 = "Hallo!". Dann würde s1.length() den Wert 6 zurückgeben, da Hallo 5 Buchstaben hat und ist ! dann das 6. Zeichen ist.

charAt(char c1)

Die Funktion gibt den char an der entsprechenden Stelle des Strings aus auf dem es angewendet wurde. Dabei wird das erste Zeichen im String durch charAt(0) zurückgegeben. Sei wieder s1 = "Hallo!", dann würde s1.charAt(4) einen char mit dem Wert 'o' zurückgeben

equals(String s2)

Die Methode equals vergleich den Inhalt des String auf dem es angewendet wird, mit einem anderen String, dem diese Methode als Argument übergeben wird und gibt einen boolean zurück. Die Methode gibt true zurück, wenn die Strings die selbe Länge haben und an jeder Stelle das gleiche Zeichen steht. String s1 = "Hallo" String s2 = "Hallo". Diese Strings sind zwei unterschiedliche Objekte wie man leicht über s1==s2 feststellen kann. Aber der Inhalt ist der selbe und daher wird s1.equals(s2) true zurückgeben.

indexOf(String s2)

Die Methode gibt die Stelle des ersten auftretens einer der Methode als Argument übergeben Zeichenfolge an in dem String an. Seien String sequenceOfLetters = "Str" String s1 = "Strings sind cool!". Dann wird s1.indexOf(sequenceOfLetters) den Wert 0 zurückgeben, welcher auch der Index ist ab dem die Zeichenkette zum ersten Mal auftritt. Es wird ein -1 als Wert ausgegeben, wenn es keinen index gibt für den der match erfüllt ist.

compareTo(String s2)

Vergleicht den String auf dem es angewendet wird mit dem String welcher der Methode als Argument übergeben wird nach der Lexikografischen Ordnung und gibt einen int-Wert zurück, welcher die Relation der Strings darstellt. Zum Beispiel gibt einen negativen Wert aus, wenn der String auf dem das angewendet wird, nach der lexikografischen Ordnung vor dem String, welcher als Argument übergeben wird, kommen würde. Sei String s1 = "abc" String s2 = "bcd", dann wird s1.compareTo(s2) einen negativen Wert ausgeben. In diesem Fall -1. Sind die Strings vom Inhalt gleich, dann gibt Methode 0 als Wert zurück

concat(String s2)

Diese Methode hängt den angegebenen String s2 an das aufrufende String-Objekt an. Sie gibt einen neuen String zurück, der die Verkettung des Originalstrings und von s2 repräsentiert.

contains(CharSequence sequence)

Diese Methode überprüft, ob die Zeichenfolge (CharSequence) sequence in der aufrufenden Zeichenkette enthalten ist. Sie gibt true zurück, wenn die Zeichenfolge gefunden wird, andernfalls false

split(String regex)

Diese Methode teilt den String um Übereinstimmungen mit dem angegebenen regulären Ausdruck regex und gibt ein Array von Strings zurück. Der Originalstring bleibt unverändert.

trim()

Diese Methode gibt eine Kopie des Strings zurück, bei dem führende und abschließende Leerzeichen entfernt wurden. Der Originalstring wird dabei nicht verändert

toUppercase()

Diese Methode gibt einen neuen String zurück, in dem alle Zeichen in Großbuchstaben konvertiert wurden. Der Originalstring wird dabei nicht verändert.
